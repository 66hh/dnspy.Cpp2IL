# dnspy.Cpp2IL
Gives possibility to open Il2Cpp builds in DnSpy! Just drag GameAssemblies.dll to dnSpy window!
# How to install
1) Download the latest release or compile it yourself
2) Copy all dlls to dnSpyInstallFolder/Extensions/
3) Enjoy!
